### Installation

1. Get a free API Key at (https://openweathermap.org/)
2. Clone the repo
3. Install NPM packages
   ```sh
   npm install axios
   npm install yargs
   ```
4. Enter your API in `CapitalWeather.js`
   ```JS
   const myApiKey = 'ENTER YOUR API KEY';
   ```
